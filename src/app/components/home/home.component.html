Esta es una pequeña aplicación que explota una API REST alimentada con datos geográficos y demográficos de todas las poblaciones de Andalucía.
La información que se puede obtener de cada población es:

Nombre de la Población, Provincia a la que pertenece, código del Instituto Nacional de Estadística, Latitud y Longitud, Altitud (en metros), Superficie (en Kilómetros cuadrados) y los Habitantes censados en 2018

It uses utility classes for typography and spacing to space content out within the larger container.
